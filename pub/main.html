BoundingBox.js

About

A js library to draw bounding boxes on images, label parts of images and annotate an image.

Visit the documentation here.

Example 1 - A Basic Use Case of the BoundingBox.js

We can use our library for computer vision related visualization, such as object detection in the streets of Hong Kong.

Click on individual bounding boxes to check out it's content.

Hide or show individual bounding boxes by clicking on the it's corresponding label in the legend located on bottom right.

If you don't like where the legend is placed you can relocate the legend simply by dragging it.

Example 2 - Set the Behavior of BoundingBox.js

You can also use the library to annotate your favorite scene from a movie.

Here we set the bounding box by default to not display the legend and not allow boundingbox onclick functionality.

Toggle the switches at the bottom to change the image's behavior.
